DLL Import
----------------------------------------------------------------------
Copy assimp-vc140-mt.dll of ToCONFIG to C:/Windows/SysWOW64/


Visual Studio - working on project
----------------------------------------------------------------------
Set the variable logl_root in /Include/learnopengl/root_directory.h
to your root directory (%GIT%/RainyNight)


Visual Studio - if includes not working
----------------------------------------------------------------------
Add the folders Libs and Include of the Repository to
Project -> RainyNight Properties -> VC++ Directories
at Library Directories and Including Directories
